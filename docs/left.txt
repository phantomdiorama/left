Setup              *LeftSetup*

You must configure Left to use the same background colour as your colorscheme. The
quickest way to find the colour values is to run:

`:highlight Normal`

Then add them to your vimrc:

```
let g:left_guibg = "#000000"
let g:left_ctermbg = "0"
```

=================================
Usage              *LeftUsage*

Left exports three commands:

:Left ............ Turn on
:LeftOff ......... Turn off
:LeftToggle ...... Toggle
